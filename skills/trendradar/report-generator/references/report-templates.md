# 周报模板

## 格式铁律（必须遵守）

1. **全文无 `---` 横线** — 板块间用双空行分隔
2. **板块标题后** 跟双空行
3. **每个主题内部** 单空行分隔段落
4. **主题之间** 双空行分隔
5. **不要用「查看原文」链接** — 来源直接在分析文本中标注（如[路透社]）
6. **不要用日报的 🔥/🆕 标记**
7. **禁止 LLM 重写格式** — 严格按此模板输出

## 周报输出模板

### Hermes趋势周报 · YYYY/MM/DD - YYYY/MM/DD

### 📰 本周头条趋势
**1. 趋势主题**
事件链 → 数据支撑 → 影响分析 → 展望 → 置信度(高/中/低+原因) → 未覆盖gap

### 🌏 外媒看华趋势
（同上结构，外文精确翻译）

### 🚀 科技数码趋势
### 📊 民生经济趋势
### 🎮 游戏市场趋势
### 📝 本周总结
跨板块交叉趋势、风险提示

### 🌱 本周意外发现
_serendipity 条目分析 + 盲点审计输出

📌 数据来源：TrendRadar 本周 N 条聚合 · 深度研究
