# 周报模板

## 格式铁律（必须遵守）

1. **全文无 `---` 横线** — 板块间用 `\n\n\n`（2个空行）分隔
2. **板块标题后** 跟 `\n\n\n`（2个空行）
3. **每个主题内部** 用 `\n\n`（1个空行）分隔各段落
4. **主题之间** 用 `\n\n\n`（2个空行）分隔
5. **不要用「查看原文」链接** — 数据来源直接在分析文本中标注（如[路透社]）
6. **不要用日报的 🔥/🆕 标记**
7. **禁止 LLM 重写格式** — 严格按此模板输出
